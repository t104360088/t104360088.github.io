使用Hexo架設自己的部落格網站

發表於 2021-04-04

身為一位將成為軟體工程師的人而言，怎麼能沒有屬於自己的技術部落格呢?因此為了紀念本部落格，第一篇文章就來說說如何快速架起部落格吧!

架設部落格從入門到放棄…，其實從頭開始要花費的時間也不少，所以本篇使用Hexo這套工具架設部落格。Hexo 是一個快速、簡單且強大的網誌框架，使用 Markdown（或其他標記語言）解析文章，而且提供眾多主題任君挑選，功能也相對完整，用於撰寫文章足以!

安裝 Node.js

Hexo 是基於 Node.js 所開發的網誌框架，因此在使用相關指令服務時，需事先安裝 Node.js，讀者可至官網下載 LTS 版本

安裝 Git 與申請 GitHub 帳號

Git 是版本控制工具，它可讓我們控管與紀錄部落格設定、文章的版本，以及利於部屬到 GitHub，我們可以下載 Git for Windows 這套為 Windows 設計的終端機來操作 Git，而 GitHub 是遠端儲存庫，可以把它想像成程式碼的雲端倉庫，我們須先至官網申請帳號

修改部落格配置

部落格的檔案結構
|檔案名稱、資料夾|說明|
|:–|:–|
| node_modules/ | 相依套件 |
| public/ | 生成部落格的靜態資源 |
| scaffolds/ | 存放架構模板，例如:文章建立後的基本架構 |
| source/ | 存放部落格所需的資源，例如:文章、圖片 |
| source/_posts | 存放部落格文章 |
| themes/ | 部落格主題樣式 |
| _config.yml | Hexo主要設定檔 |
| package.json | 相依套件說明 |

修改部落格設定

部落格可以設定的內容不少，讀者可以修改後再架起伺服器來查看效果
這裡先介紹比較基本的內容
|屬性|說明|
|:–|:–|
| title | 部落格標題 |
| author | 作者名稱 |
| language | 部落格語言 |

特別介紹一下部落格語言，我們可以在 themes/next/languages 查看 NexT 主題有支援的語言，而內部的檔案可以修改各欄位的翻譯名稱，若要將部落格設定成中文版本，需要開啟 blog/_config.yml檔，並將 language 改成 zh-TW

修改主題設定

NexT 主題內也有其內容的設定，要修改設定需要開啟 blog/themes/next/_config.yml檔
以下列出幾個比較基本的設定，只要將檔案內的欄位修改成以下的值即可

更換樣式方案、開啟暗黑模式

scheme: Mist (也有其他的方案可挑選)
darkmode: true

新增菜單選項

menu:
about: /about
tag: /tags

指定資料夾後，至 blog/source 新增對應的資料夾，並新增 index.md 檔，或是輸入以下指令建立頁面檔案

修改檔案內容
==about/index.md==

==tags/index.md==

修改側邊欄寬度

sidebar:
width: 240

修改作者頭像、修圓角

avatar:
url: /images/avatar.gif
rounded: true

新增社交資訊

social:
GitHub: 網址

修改程式碼高亮主題

codeblock:
highlight_theme: night
